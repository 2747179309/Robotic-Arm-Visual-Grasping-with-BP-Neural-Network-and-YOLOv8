# Robotic-Arm-Visual-Grasping-with-BP-Neural-Network-and-YOLOv8
Using the 幻尔xArm 2.0 robotic arm and Intel D435i depth camera, relying on YOLOv8 to visually recognize apples and accurately grasp them, with a BP neural network used for error compensation.

# Attention: D435i_yolo222n.py is used to test depth cameras and the yolov8 vision recognition model, while ik_pitch_arm_control222.py is used to test kinematic inverse solutions and robotic arm control. 
When downloading, please save them in the same folder and ensure you have the corresponding libraries and dependencies. You only need to run vison_to_arm_control_v3.py to achieve intelligent recognition and grasping.
